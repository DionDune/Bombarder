# Bombarder



Controls: 
  > Movement:

     - W/S : Up/Down
     - A/D: Left/Right
 
  > Attacks:

     - Left-Click: Place Orb
     - Right-Click: Launch Orb
     - Middle-Click: Create Wave

  > Other:

     - V: Spawn Enemy
     - I: Toggle Entity AI
     - O: Toggle Hitbox Rendering
